# 什么是 Frpc-Hub?

**Frpc-Hub** 是一款专为 **Frp** (Fast Reverse Proxy) 打造的现代化可视化管理平台。它彻底改变了传统的命令行操作方式，通过优雅的 Web 界面，为您提供简单、高效且直观的内网穿透管理体验。

Frpc-Hub 远不止于一个图形化外壳，它是您内网穿透服务的智能指挥中心。无论是集中管理多个 Frp 节点、动态配置代理规则，还是实时监控服务健康状态，它都能助您轻松应对，让复杂的网络穿透变得触手可及。

## 为什么选择 Frpc-Hub?

Frp 是目前最流行的内网穿透工具之一，但原生的 Frpc 客户端通常需要通过命令行启动，并且依赖于 `ini` 或 `toml` 配置文件。对于需要频繁修改配置或管理多个穿透服务的用户来说，存在以下痛点：

- **配置繁琐**：手动编辑配置文件容易出错且不直观。
- **缺乏监控**：无法实时直观地查看各代理服务的连接状态和流量情况。
- **管理不便**：缺少统一的管理界面，难以同时管理多个 Frpc 服务器连接。
- **重启频繁**：修改配置后通常需要手动重启服务才能生效。

**Frpc-Hub 正是为解决这些问题而生。**

## 主要特性

Frpc-Hub 提供了一站式的解决方案：

### 🖥️ 可视化服务器管理

轻松创建、编辑和删除 Frpc 服务器配置。支持同时管理连接到不同 Frps 服务端的多个实例，并且可以查看每个服务器的连接状态和地理位置信息。

### 🔌 直观的代理配置

支持通过 Web 界面图形化配置各种类型的代理（Proxy），包括：

- **HTTP / HTTPS**
- **TCP / UDP**
- **STCP / XTCP** (即将支持)

无需记忆复杂的配置参数，表单式引导让配置变得简单。

### 🔄 配置热重载

基于先进的配置管理机制，当你在 Web 界面修改代理配置后，Frpc-Hub 会自动处理配置的重新加载，**无需手动重启服务**，确保业务的连续性。

### 📊 实时状态监控

通过仪表盘实时查看：

- 服务器连接延迟
- 代理隧道在线状态

### 🎨 现代化的用户体验

- **精美界面**：基于 Radix UI 和 Tailwind CSS 构建，提供流畅、美观的操作体验。
- **深色模式**：支持亮色/深色主题切换。
- **多语言支持**：内置国际化支持（中文/英文），满足不同用户需求。

### 🚀 高性能与跨平台

- **后端**：使用 Go 语言开发，资源占用极低，运行稳定。
- **前端**：使用 React 生态，响应迅速。
- **跨平台**：提供 Windows, macOS, Linux 的预编译二进制文件，开箱即用。

## 适用场景

- **开发者**：需要将本地开发环境（如 Web 服务、数据库）暴露给公网访问或调试 Webhooks。
- **运维与极客**：管理家中或公司内网的多台设备（NAS、路由器、服务器），并在统一的面板中监控它们的状态。
- **团队协作**：快速共享本地服务给远程同事进行联调。
